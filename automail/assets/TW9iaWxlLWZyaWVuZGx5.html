Mobile friendly template
The structure is very simple. One table with several rows each having one cell.
This table is always positioned in the center.
Depending on the screen size it will expand or contract.
The table has  the class of scaletable.
Look in the <style> part of the html code. You can change the table width if you need.
The tables cells must have a defined width. In this example that is 600px.
